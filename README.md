# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn behavior using Power BI and customer subscription data.

The objective was to identify:
- Why customers leave the platform
- High-risk customer segments
- Customer lifetime patterns
- Retention improvement strategies

---

## Tools Used
- Power BI
- Excel
- Python (for data cleaning)

---

## Key Insights
- Month-to-month customers showed the highest churn
- Customers without tech support had higher churn behavior
- Early-stage customers were more likely to leave
- Electronic check users exhibited increased churn risk

---

## Business Recommendations
- Improve customer onboarding
- Promote long-term contracts
- Enhance customer support services
- Improve payment flexibility
